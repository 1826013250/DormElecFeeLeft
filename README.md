# DormElecFeeLeft

> **⚠️ 特别说明：本程序是专门针对 **河南工业大学（HAUT）** 的宿舍用电系统环境开发的。依赖于特定的数据接口，其他院校系统无法直接使用。**

DormElecFeeLeft 是一款基于 Python 和 PySide6 开发的桌面系统托盘应用程序。主要用于定时监控宿舍的剩余电量，当电量低于用户设置的阈值时，程序会自动弹出警告，提醒您及时缴费。

## 主要功能

- **系统托盘后台运行**：程序启动后将常驻在系统托盘中，不占用界面与任务栏空间。
- **自定义查询间隔**：支持自定义电量查询的时间间隔（提供从毫秒到天的多种时间单位选择）。
- **低电量警告**：当获取到的电量低于设置的警告下限时，自动弹出系统消息进行提示。
- **开机自启支持**：支持设置随系统开机自动运行。（**TODO**: 目前开机自启仅支持 Windows，macOS、Linux 支持即将到来）
- **多维度宿舍选择**：提供直观的分栏视图，支持进行校区、楼栋、楼层、寝室的多级联查与选择。

## 安装与运行

你可以通过以下两种方式运行该程序：

### 1. 从 Release 下载 (推荐)
建议直接前往本仓库的 **Release 页面** 下载已经打包好的免安装可执行文件（如 `.exe` 格式）。下载后直接双击运行即可，无需在本地配置 Python 开发环境。

### 2. 命令行源码运行 (开发者)

如果你希望通过源代码运行此程序，请确保你的系统上已经安装了 Python 环境。

首先，请进入项目根目录，通过 pip 安装必须的依赖包：

```bash
pip install -r requirements.txt
```

依赖安装完成后，在项目根目录下通过命令行运行以下命令启动主程序：

```bash
python main.py
```

### 3. 使用 Nuitka 编译 (进阶)

如果您希望自行将其编译为单文件可执行程序，我们推荐使用 [Nuitka](https://nuitka.net/)，这可以有效减小体积并提升运行效率。

在使用前请确保已安装所需依赖及 Nuitka：
```bash
pip install nuitka
```

基于不同操作系统，编译为单文件可执行程序的参考命令如下：

**Windows**（已隐藏后台控制台）：
```bash
nuitka --onefile --plugin-enable=pyside6 --windows-console-mode=disable main.py
```

**macOS** & **Linux**：
```bash
nuitka --onefile --plugin-enable=pyside6 main.py
```

## 基础使用说明

1. **初次启动配置**：初次打开软件时，系统会弹窗提示。请根据提示输入您的学号，随后在弹出的宿舍选择器中逐级选择并绑定您的宿舍号。
2. **托盘菜单操作**：右键点击右下角的系统托盘图标，可以实时查看当前剩余电量与单位，或是进行其它设置（如修改学号、重选宿舍、调整警告阈值、更改查询时间间隔等）。
3. **设置开机自启**：在右键托盘菜单中勾选“开机自启”项，即可自动配置。
